# 🌟프로젝트 소개
> **Class404** <br> 대한민국 1등 온라인 클래스 플랫폼입니다. 수업에 필요한 준비물도 구매할 수 있습니다. 해야만 하는 일에 밀려, 방법을 몰라서, 시작을 망설였다면, 사랑하는 일을 하기 위한 기반이 필요하다면, 당신이 사랑하는 일을 하며 살수 있도록 1,000명 이상의 크리에이터와 함께 취미도, 직무도, 경제적인 자유도 클래스404에서 시작하세요!
